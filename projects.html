Линейные модели

Реализовал линейную и логистическую регрессию.

Для линейной регресии есть возможность выбора метода нахождения весов: через точное матиематическое решение или же с помощью градиентного спуска. Для последнего можно выбрать как полный спуска, так и mini-batch спуск. Когда же размер batch'a равен 1 получается SGD (стохастический градиентный спуск). Реализованы L1 и L2 регуляризации. Параметры спуска задаются через шаг и условие остановки (через функцию потерь).

Алгоритм логистической регрессии может быть применен только с использованием градиентного спуска (также полного, mini-batch и SGD). Добавлена регуляризаиця. В отличие от линейной регрессии параметры градиентного спуска задаются через число итераций и шаг.

Древовидные модели

Реализовал дерево решений и случайные лес.

Классифицирующее дерево решений строится, уменьшая энтропию в подвыборках объектах. Есть возможность выбрать три критерия остановки: в каждом листе все объекты одного класса, максимальное число листьев, минимальное число объектов в листе. Значения для разделения берутся из выборки, поэтому точность может быть немного ниже sklearn'a, однако она вплотную подходит к нему. Также релизованы функция визуализации дерева и выдачи списка предикатов для каждого объекта для повышения объясняющей способности алгоритма.

Алгоритм классификации под капотом использует решающее дерево из sklearn для сохранения высокой скорости. Может работать на несбалансированных классах. Для сэмплирования подвыборок для каждого из деревьев используется бутстрап. Метрики сравнимы с sklearn, время работы может быть чуть дольше. Так же на данный момент не реализована совместимость с GridSearchCV.
